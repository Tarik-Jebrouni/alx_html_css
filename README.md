## 🎯 Features

- Fully responsive layout that adjusts gracefully for screen widths **480px and below**
- Clean, centered content with a **maximum width of 1000px**
- Interactive elements styled to match design expectations:
  - **Hover/Active link color**: `#FF6565`
  - **Button hover/active**: `opacity: 0.9`
- Mobile-first design principles applied
- Accessible and semantic HTML structure

## 📱 Responsive Design

| Screen Size | Behavior |
|-------------|----------|
| ≥ 481px     | Desktop layout |
| ≤ 480px     | Switches to mobile version |

Responsive behavior was implemented using media queries to ensure optimal viewing on all devices.

## 📐 Design Source

- The design was provided as a **Figma file**.
- Make sure to “Duplicate to your Drafts” in Figma to access all layers and design specs.
- Some spacing and dimension values were rounded from float values for practical implementation.
